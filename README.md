# Asthma Predictive Models Comparison 🫁📊

## Overview
This project focuses on building and evaluating machine learning models to predict asthma risk. By leveraging healthcare data sourced from Kaggle, the project compares multiple classification algorithms to identify the most accurate and reliable predictive model.

## 🛠️ Tools & Technologies
* **Environment:** AI Studio
* **Algorithms:** Random Forest, Decision Tree, Naive Bayes
* **Data Source:** Healthcare Dataset (Kaggle)

## 🚀 Key Contributions
* **Model Development:** Built predictive models using Random Forest, Decision Tree, and Naive Bayes with healthcare data from Kaggle.
* **Model Evaluation:** Evaluated model performance for risk prediction, identifying Random Forest as the optimal algorithm with 99.57% accuracy and a 99.11% F-measure.
